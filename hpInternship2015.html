What is the Sizing Process?

Sizing is the formulation of a budget for a particular project and getting approval to proceed on the project.

Components of a Sizing

  1. A project manager creates a request for the sizing of a project
  2. Request is sent to app teams related to the project
  3. Each team completes a sizing
  4. All sizing information is consolidated and a budget is created

Problems with the Old Sizing Tool

  • Inefficient (confusing, database maintenance, multiple SharePoints)
  • Lack of standardization
  • Difficult to consolidate
  • Heavy burden on user
